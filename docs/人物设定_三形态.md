# 主角人物设定 —— 三个年龄形态

> 来源:策划提供的《人物文字描述.doc》(2026-07-12 微信传入,已解析归档)。
> **核心**:主角是同一个人物的三个年龄形态,对应本作时间循环机制 `GameState.Era`:青年 Young / 中年 Middle / 老年 Old。
> 主题呼应:一个探险家从"自信鲁莽"→"妥协精密"→"衰弱释然"的一生,最终发现寻找的不是神明而是被遗忘的勇气。

---

## 青年形态(Youth · Era.Young):自信鲁莽的机械探险家

- **视觉基调**:敏捷、充满张力、带有扩张性的轮廓。色彩饱和度最高,与神庙的阴暗形成鲜明对比。
- **头部**:短发,额头上推着一副防风/探险护目镜,露出几根凌乱不羁的刘海。眼神明亮,瞳孔玛瑙色或棕色。面部表情透着一丝不知天高地厚的莽撞。
- **躯干**:穿着耐磨的无袖帆布探险夹克,露出结实的手臂。胸前绑着结实的攀岩安全带和黄铜卡扣。
- **手臂/双手**:右手佩戴一只厚重的**机械辅助护臂**,带有粗糙的齿轮和外露的线缆——这是他能用蛮力破坏关卡障碍的视觉依据。(⚠待确定:护臂为原始装备 / 关卡资源)
- **腰部**:厚皮带上挂着几个功能性粗布口袋,腰带上卡着登山绳索。
- **腿部与脚部**:修身工装裤,裤脚扎进厚底登山靴。**待机和跑动动作要轻盈、有跳跃感。**
- **核心剪影**:修长倒三角形,重心偏上,充满爆发力。**身高最高。**

## 中年形态(Middle-Age · Era.Middle):妥协与精密的钟表匠

- **视觉基调**:沉稳、内敛、收缩的轮廓。色彩以黄铜色、深棕色和灰蓝色为主,体现现实的重压与绝对的规则感。
- **头部**:头发梳理得一丝不苟(可能带些许灰白)。鼻梁上架一副带可翻折放大镜片的精密机械眼镜(用于处理精密仪器)。神情严肃、紧绷,眉头深锁。
- **躯干**:修身衬衫,外罩一件满是油污和岁月痕迹的深色皮革工匠围裙。因长年伏案,背部有轻微佝偻。
- **手臂/双手**:挽起袖子,小臂结实。不再戴粗犷护臂,改戴半指皮手套,手指修长稳定,适合操作精密解谜机关。
- **腰部**:围裙胸前和腰部插满各种精密工具(镊子、游标卡尺、小型螺丝刀)——这是他能在工具栏进行**道具组合**的视觉依据。
- **腿部与脚部**:黑色战术裤(略显陈旧),硬底皮鞋。**站姿绝对稳定,双脚像钉在地上,底盘极稳。**
- **核心剪影**:长方形,重心在正中。

## 老年形态(Old Age · Era.Old):释然与衰弱的寻神者

- **视觉基调**:脆弱、佝偻、边缘模糊的轮廓。脚步缓慢,色彩饱和度低,极其黯淡。
- **头部**:稀疏白发(可能戴帽子),脸上布满极深的皱纹。佩戴一副老旧而过大的老花镜。瞳孔变暗蓝,眼睛虽浑浊,但透出能看破岁月迷雾的深邃光芒。
- **躯干**:一件过于宽大、褪色的旧大衣(仿佛躯体已萎缩,撑不起衣服),显得极其单薄无力。
- **手臂/双手**:枯瘦的双手,待机时会微微颤抖;**这种颤抖只有在举起特定道具(如提灯)时才勉强稳定下来。**左手撑一根光秃秃的拐杖。
- **腰部**:没有多余工具,只有一条松垮的旧皮带。
- **腿部与脚部**:藏在大衣里若隐若现的裤子,一双破旧柔软的布鞋。**站姿极度佝偻,移动时是擦着地面的拖拽步态。**
- **核心剪影**:弯曲的问号形状,重心极低且不稳,视觉上极度弱势。

## 对程序/系统层的启示(A 关注)

1. **三形态 = 同一 PlayerController + 三套外观/动画**。`GameState.Era` 切换时,换 sprite 组、换行走动画、换移动速度与手感:
   - Young:移动快、轻盈跳跃感 → `moveSpeed` 最高
   - Middle:沉稳,站姿极稳 → 中速
   - Old:拖拽步态、缓慢 → `moveSpeed` 最低,可加待机颤抖动画
2. **每形态有独特"交互能力/道具依据"**,可能与谜题解法绑定:
   - Young:机械护臂 → 蛮力破坏障碍类交互
   - Middle:精密工具围裙 → 工具栏**道具组合**类谜题
   - Old:提灯稳定手抖 → 举灯照明类交互
   → 建议 `InteractableBase` 或谜题预留"当前 Era 是否满足能力要求"的判定钩子。
3. **剪影/身高不同**(倒三角→长方形→问号,身高从高到矮):美术出图与碰撞体/脚底锚点需按形态区分,`PlayerController` 深度排序按脚底 Y 值仍通用。
